Packet Sniffer and Spoofer

This project repository houses an IP packet sniffer program, a packet spoofer program, and a working server simulation to test said programs. The packet sniffer can capture and analyze internet protocol packets moving through a network. This tool has many applications for network troubleshooting, monitoring, and security analysis. The packet spoofer is capable of forging network packets and generating false traffic. This tool has numerous applications, especially in the cybersecurity field. Packet spoofing uses include but are not limited to network simulation, intrusion detection testing, penetration testing, load balancing, and network traffic redirection. I made this project using C++. Working on this project expanded my familiarity with network protocol, network interfacing, and cybersecurity.
